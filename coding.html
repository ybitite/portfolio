REST API Integration with Retrofit
Demonstrates API integration with Retrofit, including error handling and data parsing. 

        // Setting up Retrofit instance
        Retrofit retrofit = new Retrofit.Builder()
            .baseUrl("https://api.example.com/")  
            .addConverterFactory(GsonConverterFactory.create()) 
            .build();

        // Defining the API service
        public interface ApiService {
            @GET("items")  
            Call> getItems();  
        }

        // Making an API call
        ApiService service = retrofit.create(ApiService.class);  
        service.getItems().enqueue(new Callback>() {  
            @Override
            public void onResponse(Call> call, Response> response) {
                if (response.isSuccessful()) {
                    List items = response.body();  
                    // Handle the received data
                } else {
                    Log.e("API Error", "Error code: " + response.code());  
                }
            }

            @Override
            public void onFailure(Call> call, Throwable t) {
                Log.e("API Failure", t.getMessage());  
            }
        });

Local Database Management with Room
Shows how to manage a local database using Room, including DAO and entity setup. 

        // Entity class for database
        @Entity(tableName = "items")  
        public class Item {
            @PrimaryKey(autoGenerate = true)  
            private int id;

            private String name;  
            private double price;  
        }

        // DAO Interface for database operations
        @Dao
        public interface ItemDao {
            @Insert  
            void insert(Item item);

            @Query("SELECT * FROM items")  
            List getAllItems();
        }

        // Database class for Room database configuration
        @Database(entities = {Item.class}, version = 1)  
        public abstract class AppDatabase extends RoomDatabase {
            public abstract ItemDao itemDao();  
        }
